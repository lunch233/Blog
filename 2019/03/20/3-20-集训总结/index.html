给你一个长为 nnn 的序列 ddd ， qqq 次询问。一个区间 [l,r][l,r][l,r] 的价值为val(l,r)=(∑i=lrdi)×(r−l+1)val(l,r)=(\sum\limits_{i = l}^rd_i)\times(r-l+1)val(l,r)=(i=l∑r​di​)×(r−l+1)，每次询问一个区间价值大于等于 xxx 的子区间中，子区间最大的 ddd 最小是多少。

n,q≤3×105,di≤107,1≤li≤ri≤n,xi≤1018n,q\le3\times10^5,d_i\le 10^7,1\le l_i \le r_i \le n,x_i \le 10^{18}n,q≤3×105,di​≤107,1≤li​≤ri​≤n,xi​≤1018。

Expected Score：8

Really Score：23

莫名 O(n2log⁡n)O(n^2\log n)O(n2logn) 过了 n≤5000n \le 5000n≤5000 ，考试的时候一直刚根号做法没有刚出来，最后只有暴力分。

实际上这道题对于一个询问 (L,R,x)(L,R,x)(L,R,x) ，我们需要的区间[a,b][a,b][a,b]只需要满足下列条件中的一个：

    1. a=La=La=L 或 b=Rb = Rb=R 。
    1. db,da>max⁡j=a+1bdjd_b,d_a>\max\limits_{j = a + 1}^b d_jdb​,da​>j=a+1maxb​dj​。

第一类区间我们只需要最小化区间长度，例如 a=La = La=L 我们只需要找到一个最大的 ppp 满足 val(L,p)≥xval(L,p)\ge xval(L,p)≥x，只需要二分 ppp 的位置即可，b=Rb = Rb=R 时同理。

第二类区间实际上总数只有 nnn 个，我们可以用单调栈来找到他们，然后我们把这 nnn 个区间价值和询问的 xxx 一起从大到小排序，用一个线段树维护每个点作为左端点时区间最大的 ddd 最小是多少，假如有跨过右端点的区间那第一类区间就是这种区间的一个子区间，不会影响答案，因为从大到小排序所以区间价值的限制肯定满足。

总复杂度 O((n+m)log⁡n+m)O((n+m)\log{n + m})O((n+m)logn+m)。

求一个长度为 nnn ，出现了 kkk 种字符，字典序最小的字符串，满足串的每个前缀与后缀出现的字符两两不同。

n≤105,k≤26n \le 10^5,k \le 26n≤105,k≤26。

Expected Score：58

Really Score：14

考试的时候判无解的一种情况输出了小写，直接把 n≤20n \le 20n≤20 的 444444 分全丢了。

先写一发暴力，很容易发现字符串足够长时就是在前面无限加 aaa ，那么我们只要找到刚好有解时的字符串的规律即可：

    1. n=1,k=1,ans=an=1,k=1,ans=an=1,k=1,ans=a。

    2. n=5,k=2,ans=aabaan = 5,k = 2,ans=aabaan=5,k=2,ans=aabaa。

    3. n=12,k=3,ans=aababacbabaan = 12,k = 3,ans=aababacbabaan=12,k=3,ans=aababacbabaa

设 B=ba,D=dcbaB=ba,D=dcbaB=ba,D=dcba ，按这种方法类似定义，那么上面的东西变成了：

    1. n=1,k=1,ans=An = 1,k = 1,ans=An=1,k=1,ans=A 。
    2. n=5,k=2,ans=AABAn = 5,k = 2,ans=AABAn=5,k=2,ans=AABA 。
    3. n=12,k=3,ans=AABBCBAn = 12,k = 3, ans = AABBCBAn=12,k=3,ans=AABBCBA。

大胆猜想可以发现， k=4k = 4k=4 的答案是 AABBCCDCBAAABBCCDCBAAABBCCDCBA。

答案的规律已经很显然了，那么直接算就好了，复杂度 O(n+k)O(n + k)O(n+k)。
